# ♻️ Waste Classification using Image Processing

## 📌 Project Overview
This project uses Machine Learning / Deep Learning to classify waste images into different categories for better waste management and recycling.

## 🚀 Features
- Image-based waste classification
- Trained model on labeled dataset
- Accurate prediction of waste categories

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy, Pandas
